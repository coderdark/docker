# Docker


| Description                           | Command                                                                              |
|---------------------------------------|--------------------------------------------------------------------------------------|
| To build the image                    | docker build -t <IMAGE_NAME> .  (-t to provide a name for the image)                 |
| To run the image                      | docker run -d --rm -p 80:8000 --name <CONTAINER_NAME> <IMAGE_NAME> 
                                                                                                                               |
